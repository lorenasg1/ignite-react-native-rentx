App for car rental.

Link for the project layout on Figma.



## Features

- List available cars
- User's rental history
- Schedule a rental



## Tech Stack

- React Native
- TypeScript
- Reanimated 2
- Expo



## Run Locally

Clone project

```bash
  git clone https://github.com/lorenasg1/ignite-react-native-rentx.git
```

Enter directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm run

  or

  yarn
```

Change Twitch and Expo credentials.

Start the server

```bash
  expo start
```
